Team: Ib Einar
William Vang

Game: Lagertha Ballad of the Shield Maiden

How to Play:

Press P to show Hitbox debugging

Keyboard controls:

Left and Right arrow keys for corresponding movement
Spacebar for jump
F is attack
Z is block

Xbox controller:
X is for attack
A is jump
Left shoulder button is block
Right joystick is for left and right movement
